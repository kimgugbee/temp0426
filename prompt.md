# 영화 퀴즈 웹앱 제작 프롬프트

## 1. 페르소나 설정
너는 React와 Tailwind CSS에 정통한 시니어 개발자야. 사용자 경험(UX)을 최우선으로 생각하며, 깔끔하고 유지보수가 쉬운 코드를 작성해.

## 2. 프로젝트 요구사항
- **목표**: 사용자가 영화 포스터나 줄거리를 보고 제목을 맞히는 퀴즈 앱 제작
- **핵심 기능**:
  - 시작/퀴즈/결과 3단계 화면 전환
  - 실시간 점수 계산 및 진행률 표시 바(Progress Bar)
  - Framer Motion을 활용한 부드러운 화면 전환 애니메이션
- **기술 스택**: React, Tailwind CSS, Framer Motion, Lucide-react(아이콘)

## 3. 디자인 가이드
- **테마**: 시네마틱 다크 모드 (Slate-900 배경)
- **포인트 컬러**: 황금색(Gold)과 강조용 레드
- **UI 스타일**: 카드 기반 레이아웃, 버튼 호버 및 클릭 애니메이션 필수

## 4. 요청 사항
먼저 전체적인 프로젝트 구조를 제안해주고, 퀴즈에 사용할 샘플 데이터(JSON) 형식을 정의해줘. 그 다음 메인 컴포넌트부터 단계별로 코드를 작성해줘.
